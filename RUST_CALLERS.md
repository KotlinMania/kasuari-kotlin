**Cross-repo only.** Imports between crates inside this same repo are omitted ‚Äî those are intra-repo refactoring concerns, not the cross-project porting concern this file exists to surface.

**Cargo-resolved.** Counts here honor `package = "..."` renames, `workspace = true` inheritance, and `[target.<cfg>.dependencies]`. Stdlib (`std`, `core`, `alloc`) and intra-crate sub-module references are excluded.
